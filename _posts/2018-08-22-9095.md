https://www.acmicpc.net/problem/9095

```python3
N = int(input())
count = [ int(input()) for _ in range(N)]
d = [0] * (max(count)+1)
def solving(n):
    d[1] = 1
    d[2] = 2
    d[3] = 4
    for i in range(4, n+1):
        d[i] = d[i-1] + d[i-2] + d[i-3]
    return d[n]
for i in count:
    print(solving(i))

```
d 에는 i를 1,2,3으로 만들 수 있는 가지 수를 저장한다.
