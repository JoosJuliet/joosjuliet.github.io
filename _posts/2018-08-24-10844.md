https://www.acmicpc.net/problem/10844

``` python
n = int(input())
d = [[0]*(n+1) for _ in range(10)]

for i in range(1, 10):
    d[i][1] = 1

for i in range(1, n+1):
    for j in range(0, 10):
        if j-1 >= 0:
            d[j][i] += d[j-1][i-1]
        if j+1 <= 9:
            d[j][i] += d[j+1][i-1]
ans = 0
for i in range(0,10):
    ans += d[i][n]
print(d%1000000000)
```
