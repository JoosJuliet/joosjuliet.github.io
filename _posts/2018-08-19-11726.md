https://www.acmicpc.net/problem/11726

``` python
N = int(input())
d = [0] * (N+1)

d[0] = 1
d[1] = 1

for i in range(2, N+1):
  d[i] = d[i-1] + d[i-2]

print(d[N]% 10007)
```
