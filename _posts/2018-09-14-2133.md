문제: https://www.acmicpc.net/problem/2133
``` python
import sys
n = int(sys.stdin.readline())
d = [0]*(n+1)

d[0] =1

for i in range(2,n+1,2):
    d[i] = 3*d[i-2]
    for j in range(0,i-4+1,2):
        d[i] += 2*d[j]

print(d[n])

```
