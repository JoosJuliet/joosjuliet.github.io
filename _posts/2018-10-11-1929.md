문제:
https://www.acmicpc.net/problem/1929

``` python
import sys
read = lambda : sys.stdin.readline().strip()

n, m = map(int, read().split())
check = [False]*(m+1)
check[0] = check[1] = True
i = 2
while i * i <= m:
    if check[i] == True:
        i += 1
        continue
    j = i + i
    while j <= m:
        check[j] = True
        j += i
    i += 1

for k in range(n, m+1):
    if check[k] == False:
        print(k)
```
