문제:
https://www.acmicpc.net/problem/1676
``` python

import sys
read = lambda : sys.stdin.readline().strip()

n = int(read())

ans = 0
i = 5
while i <= n:
    ans += n//i
    i *= 5

print(ans)
```
n을 100으로 볼 때
5의 배수인 것의 개수를 더한다.
또 25의 배수인 것들은 5의 배수일 때 하나로 세지기 때문에
i에 5를 곱하고 그것의 개수만큼 한번 더 더해야한다.
