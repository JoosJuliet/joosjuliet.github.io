문제:
https://www.acmicpc.net/problem/11653

``` python
import sys
read = lambda : sys.stdin.readline().strip()
n = int(read())

i= 1
while i*i <= n:
  i +=1
  while True:
    if n % i == 0:
      print(i)
      n //= i
    else:
        break

if n > 1:
  print(n)
# 미쳐 나누어 떨어지지 않은 것을 알려준다
```
