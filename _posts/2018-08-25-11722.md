https://www.acmicpc.net/problem/11722

``` python

n = int(input())
d = list(map(int, input().split()))
d.insert(0,0)
sequence = [1]*(n+1)
# sequence[i]는 i를 포함한 감소하는 수열중 가장 긴 수열

for i in range(n+1):
    for j in range(i):
        if d[j] > d[i] and sequence[i] < sequence[j]+1:
            sequence[i] = sequence[j]+1
print(max(sequence))
```
