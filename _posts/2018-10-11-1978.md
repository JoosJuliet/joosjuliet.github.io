문제링크 : https://www.acmicpc.net/problem/1978

어떤 수 N이 소수인지 아닌지 판별하는데 걸린 시간 복잡도 : O(루트 N)
``` python
input()
a = list(map(int ,input().split()))
count=0

def is_prime(x):
    if x < 2 :
        return False
    i = 2
    while i*i <= x:
        if x % i == 0:
            return False
        i += 1

    return True

for i in a:
    if is_prime(i):
        count += 1

print(count)
```
