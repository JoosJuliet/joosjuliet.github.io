문제: https://www.acmicpc.net/problem/2331

``` python
import sys
read = lambda : sys.stdin.readline().strip()

a, p = map(int, read().split())
d = []
while a not in d:
    d.append(a)
    a=sum([int(i)**p for i in str(a)])
print(d.index(a))

```
